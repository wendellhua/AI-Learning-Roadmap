## 🌟 Why Learn About AI Hardware? | 为什么要了解AI硬件？

AI is like a supercar 🏎️:
AI 就像一辆超级跑车 🏎️：

- **GPU** = Engine → Determines how fast it runs
- **GPU** = 发动机 → 决定跑得多快

- **Memory** = Fuel tank → Determines how far it can go
- **内存** = 油箱 → 决定能跑多远

- **CPU** = Driver → Coordinates all components
- **CPU** = 驾驶员 → 协调所有部件

Without a good "engine", even the smartest AI can't run!
没有好的"发动机"，再聪明的AI也跑不起来！
