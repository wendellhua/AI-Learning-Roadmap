## 🌟 What is Prompt Engineering? | 什么是提示工程？

Prompt engineering is like:
提示工程就像是：

- 🎭 **A director guiding actors | 导演指导演员** - Telling AI how to "perform" | 告诉AI该怎么"表演"
- 📝 **An outline for an essay | 写作文的提纲** - Giving AI a clear direction | 给AI一个清晰的方向
- 🗣️ **Communicating with an expert | 和专家沟通** - Asking the right questions gets good answers | 问对问题才能得到好答案

Same AI, different prompts, vastly different results!
同样的AI，不同的提示词，效果天差地别！

## 🎮 Prompt Comparison Examples | 提示词示例对比

### ❌ Bad Prompt | 差的提示词
```
Write an article
写一篇文章
```

### ✅ Good Prompt | 好的提示词
```
You are a senior tech blogger skilled at explaining complex concepts 
in easy-to-understand language.
你是一位资深的科技博主，擅长用通俗易懂的语言解释复杂概念。

Please write a popular science article about "What is a Large Language Model":
请写一篇关于"什么是大语言模型"的科普文章：

- Target readers: 12-year-old middle school students
- 目标读者：12岁的中学生
- Word count: 800-1000 words
- 字数：800-1000字
- Style: Relaxed and interesting, use examples from daily life
- 风格：轻松有趣，多用生活中的例子
- Structure: Introduction → Principles → Applications → Summary
- 结构：引言→原理→应用→总结
```
