## 🌟 什么是提示工程？

提示工程就像是：
- 🎭 **导演指导演员** - 告诉AI该怎么"表演"
- 📝 **写作文的提纲** - 给AI一个清晰的方向
- 🗣️ **和专家沟通** - 问对问题才能得到好答案

同样的AI，不同的提示词，效果天差地别！

## 🎮 提示词示例对比

### ❌ 差的提示词
```
写一篇文章
```

### ✅ 好的提示词
```
你是一位资深的科技博主，擅长用通俗易懂的语言解释复杂概念。

请写一篇关于"什么是大语言模型"的科普文章：
- 目标读者：12岁的中学生
- 字数：800-1000字
- 风格：轻松有趣，多用生活中的例子
- 结构：引言→原理→应用→总结
```
